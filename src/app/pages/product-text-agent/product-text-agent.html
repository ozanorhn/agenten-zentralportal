edit_note Produkttext Agent

Produkt-URL rein. SEO-Text raus.

Der Agent sendet genau eine JSON-Anfrage an den n8n-Test-Webhook `/webhook-test/seo-product-description`. Du waehlst entweder eine Produkt-URL oder eine Bild-URL und kannst optional Hinweise fuer die KI mitschicken.

Pflichtfeld: genau eine Variante verwenden, also entweder `productUrl` oder `imageUrl`.

Optional, aber hilfreich. Besonders bei einer reinen Bild-URL wird der Produkttext dadurch meist praeziser.
